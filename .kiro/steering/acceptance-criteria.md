# Acceptance Criteria

All code changes MUST satisfy these requirements before being considered complete:

1. **Test Suite**: `npm run test`
   - All existing tests must pass
   - New functionality requires corresponding tests

2. **Code Formatting**: `npm run format`
   - Consistent formatting across all files

3. **Build Process**: `npm run build`
   - Successful compilation for all entry points (main, v3, v4)
   - No build warnings or errors
